# House( 🚧Under Construction🚧)

> 取名House，意为记录自己内心的想法，以及一些事项。

- 记录日常酸甜苦辣（情绪模块）
- 记录待办事项（TODO模块）
- MindMap功能（暂未开始）
- 流程图功能（暂未开始）
- 定时任务（暂未开始）
- 段时间内系统自动进行总结（暂未开始）
